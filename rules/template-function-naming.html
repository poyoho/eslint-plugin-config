# 模板函数命名规范 (template-function-naming)

方法命名-操作行为以行为标识做开头，且方法名前面写上对应注释，操作行为不能被其它方法直接调用。

# 事例

此规则的错误代码示例：

<template>
  <div>
    <input @enter="enterInputName">
    <a @click="clickSubmit">提交</a>
  </div>
</template>

<script>
export default {
  methods: {
    // 点击-提交
    clickSubmit () {},

    // 回车-提交
    enterInputName () {
      this.clickSubmit() // 错误写法
    },
  }
}
</script>

此规则的正确代码示例：

<template>
  <div>
    <input @enter="enterInputName">
    <a @click="clickSubmit">提交</a>
  </div>
</template>

<script>
export default {
  methods: {
    // 点击-提交
    clickSubmit () {
      this.toSubmit()
    },

    // 回车-提交
    enterInputName () {
      this.toSubmit()
    },

    toSubmit () {}
  }
}
</script>
